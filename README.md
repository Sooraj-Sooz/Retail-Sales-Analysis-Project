# Sales Analysis Project

An end-to-end sales analytics project focused on analyzing customer behavior, product performance, payment trends, shipping methods, and seasonal revenue patterns using SQL, Python, Excel, and Power BI.

## Project Objectives
- Analyze customer and product performance
- Identify revenue trends and seasonal patterns
- Evaluate payment and shipping behavior
- Generate actionable business insights and recommendations

## Tools & Technologies
- SQL
- Python (Pandas, NumPy, Plotly)
- Microsoft Excel
- Power Query
- Power BI

## Key Features
- Data cleaning and transformation
- Exploratory Data Analysis (EDA)
- KPI and trend analysis
- Interactive Power BI dashboard
- Customer segmentation analysis
- Revenue and operational insights

## Key Insights
- 50+ age group generated the highest revenue
- Smartphones were the top-performing product category
- Non-members contributed more revenue than members
- Q4 revenue showed a noticeable decline
- Add-on sales contribution was very low
