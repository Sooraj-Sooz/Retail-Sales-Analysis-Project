# 📊 Sales Analysis Project | SQL + Python + Power BI + Excel

## 📌 Project Overview
This project focuses on analyzing customer behavior, product performance, payment trends, shipping methods, and seasonal revenue patterns using SQL, Python, Excel, and Power BI.

The goal of this project was to perform end-to-end sales analysis by cleaning and transforming raw data, identifying business trends, evaluating KPIs, and building interactive dashboards to generate actionable business insights and strategic recommendations.

# 📊 Key Performance Insights

| KPI | Insight |
|------|------|
| Top Customer Segment | 50+ Age Group |
| Highest Revenue Product | Smartphones |
| Highest Revenue Source | Non-Members |
| Peak Revenue Months | May – August |
| Lowest Performing Area | Add-on Sales (1.92%) |

---

# 📈 Dashboard Insights

- 50+ age group generated the highest overall revenue
- Smartphones led both revenue and units sold
- Non-members contributed significantly higher revenue than loyalty members
- PayPal and Credit Card were the most preferred payment methods
- Same Day and Expedited shipping showed higher-value transactions
- Q4 revenue showed a noticeable decline compared to earlier months
- Add-on products contributed very low overall revenue
